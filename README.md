# sportApp
Small project for local sport clubs.
Project Description

Developed a multi-page sports and fitness website featuring different sports, exercise complexes.

Responsibilities included designing and structuring pages for a clean and user-friendly layout, uploading and organizing content, and optimizing the site for responsiveness and speed. The website consists of ten interlinked pages, providing a professional and visually appealing experience for users.

This project demonstrates practical skills in web design, content management, and collaborative development, highlighting the ability to deliver a functional and visually engaging website.
